# Daily Planner

Modern shared planner frontend built with React and Vite.

## Frontend

Create `.env`:

```bash
VITE_API_BASE_URL=http://localhost:4000
```

Run:

```bash
npm run dev
```

The backend is maintained separately in the `Daily_Planner_BE` repository.

## Vercel

Frontend:

- Build command: `npm run build`
- Output directory: `dist`
- Set `VITE_API_BASE_URL` to your deployed backend URL
